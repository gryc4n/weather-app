# Weather App

A responsive weather application built with HTML, CSS, and JavaScript that allows users to search for any city and view real-time weather information using a public weather API.

## Preview

![Weather App Preview](preview.png)

## Features

- Search weather by city name
- Real-time weather data
- Current temperature
- Weather condition and icon
- Feels like temperature
- Humidity
- Wind speed
- Atmospheric pressure
- Error handling for invalid city names
- Responsive design

## Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- REST API
